## Hierarchie pravdy

Pokud si dva dokumenty odporují, platí tato priorita (od nejvyšší):

1. `brandbook.css` — tokeny (barvy, fonty, spacing)
2. `Switch Brand Book.html` — vizuální brand book
3. tyto MD soubory — textová doplňková dokumentace
4. ostatní HTML soubory — aplikace a příklady

Když najdeš rozpor, oprav nižší vrstvu, ne vyšší.
